## Installation

Clone repo

```
git clone https://github.com/arnoxldq/Gentoo-Dotfiles.git
cd Gentoo-Dotfiles
```

Install GNU stow

Gentoo:
```
emerge app-admin/stow
```

Apply configs

```
stow hypr kitty waybar wofi gtk-3.0
```
